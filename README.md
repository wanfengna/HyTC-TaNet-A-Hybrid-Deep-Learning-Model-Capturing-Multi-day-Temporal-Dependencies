# HyTC-TaNet

Code and data for the research paper:
**"HyTC-TaNet: A Hybrid Deep Learning Model Capturing Multi-day Temporal Dependencies for Daily Mean Air Temperature Estimation with Spatial Applicability Analysis"**

## Overview
This repository contains the source code and sample data used in our study to estimate daily mean air temperature using the HyTC-TaNet model.
